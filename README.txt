Aquarius Equipment Manager – schnelle MVP-Version

Start:
1. ZIP entpacken oder direkt bei Netlify hochladen.
2. index.html öffnen.

Zugänge:
Gerätewart: wart / aquarius
Mitglied: mitglied / aquarius

Enthalten:
- Login mit Rollen
- Mitgliederverwaltung
- Equipmentverwaltung
- Reservierungsanfragen
- vollständige und teilweise Genehmigung
- Ablehnung und Stornierung
- Historie
- iPhone-optimiertes Blau-Weiß-Design

Hinweis:
Diese Version speichert Daten lokal im jeweiligen Browser. Für gemeinsame Nutzung auf mehreren Geräten muss als nächster Schritt Supabase angebunden werden.
